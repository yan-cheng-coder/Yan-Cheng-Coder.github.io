P1002 学历史导致的

描述

yummy 对干支纪年法有一些了解，比如他知道 2025 年是乙巳（yisi）年。干支纪年法是一种用十天干、十二地支来表示年份的方法。十天干的拼音依次是 jia, yi, bing, ding, wu, ji, geng, xin, ren, gui，而十二地支的拼音依次是 zi, chou, yin, mao, chen, si, wu, wei, shen, you, xu, hai。具体地，每一个年份都对应一个天干和一个地支，计算规则如下：
·1984 年的天干是 jia，地支是 zi。
·之后每过一年，天干和地支就按顺序变成去年的天干、地支的后继。
·特别地，gui 的后继是 jia，hai 的后继是 zi。
干支纪年法就是把天干和地支的拼音拼起来，如 1984 年对应 jiazi。不难验证，在 1984∼2043年这 60 年里，公元年份对应的干支纪年法两两不同。给出一个干支纪年法，请输出其对应的、在1984∼2043 年里对应的公元年份。
由于 yummy 怕你瞎蒙，所以他会问你 T 个问题。

输入

输入的第一行有一个正整数 T（1≤T≤100），表示问题的数量。之后 T 行，每行有一个干支纪年法的拼音。

输出

对于每个问题，输出一行一个正整数，表示 1984∼2043 年里的年份。

数据范围

Time Limit (Java / Others)
2000 / 1000 MS
Memory Limit (Java / Others)
524288 / 524288 K

P1006 学博弈论导致的

描述

桌上有 r 块红宝石、b 块蓝宝石和 m 个宝盒。Alice 和 Bob 要轮流操作，Alice 先手。操作结束后，桌面清空的玩家获胜。操作后若桌面自动出现物品，出现过程也看作操作的一部分。
每次操作，两人可以选择下列操作之一：
·拿走 k 块红宝石（1≤k≤3）。
·把一块蓝宝石变成红宝石。
·拿走 1 块蓝宝石，然后拿走 0 或 1 块红宝石。
·拿走 2 块蓝宝石，然后桌面上自动出现一块红宝石。
·拿走一个宝盒说”蓝星“，然后桌面上自动出现一块蓝宝石。
·拿走一个宝盒说”赤石“，然后桌面上自动出现一块红宝石。
·拿走一个宝盒说”共生“，然后桌面上自动出现一块红宝石和一块蓝宝石。
·注意，拿走宝盒后，必须说上述三个词之一。
给出 r,b,m，假设 Alice 和 Bob 都是最聪明的，请判断谁会获胜，或报告游戏无法结束。

输入

本题共有多组数据。第一行为一个正整数 T(1≤T≤105)T，表示数据组数。之后 T 行，每行有三个自然数 r,b,m，表示一组数据的输入。保证 0≤r,b,m≤109，并且
r,b,m 不全是 0。

输出

对于每组数据，输出一行一个字符串表示答案：
·若 Alice 获胜输出 Alice。
·若 Bob 获胜输出 Bob。
·若游戏无法结束输出 Tie。
